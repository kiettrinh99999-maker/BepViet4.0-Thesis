# Backend Overview
This project is a backend system built with Laravel following the MVC architecture.
It provides RESTful APIs with authentication, token-based authorization, and user management.
The system focuses on clean code structure, separation of concerns, and secure data handling.
# Features
- MVC architecture (Model – View – Controller)
- User authentication and authorization
- Token-based authentication (JWT / Sanctum)
- User management (CRUD)
- RESTful API design
- Database modeling and migrations
